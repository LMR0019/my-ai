## 1. 功能概述

账号管理模块用于管理系统用户账号。模块包含：

**列表页** — 账号管理入口：
- 查看所有账号列表
- 统计各端口账号数量
- 搜索筛选账号
- 启用/禁用账号
- 快速操作入口

**新建/编辑弹窗** — 账号信息管理：
- 账号基本信息
- 归属机构与角色配置

**重置密码弹窗** — 密码重置：
- 重置账号密码为123456

**删除确认弹窗** — 删除账号：
- 确认删除账号

│                                                             │

└─────────────────────────────────────────────────────────────┘

### 7.3 归属机构配置

```
┌─────────────────────────────────────────────────────────────┐
│ 选择归属机构                                      [×]     │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│ 归属机构*：[肇庆市应急管理局▼]                              │
│                                                             │
│ 角色*：[超级管理员▼]                                       │
│                                                             │
│ [取消] [确认]                                               │
└─────────────────────────────────────────────────────────────┘
```

### 7.4 字段说明

| 字段 | 必填 | 说明 | 校验规则 | 新建 | 编辑 |
|------|------|------|---------|------|------|
| 账号/用户名 | 是 | 系统登录唯一标识 | 字母数字，唯一 | 是 | 否 |
| 真实姓名 | 是 | 用户真实姓名 | 最大50字符 | 是 | 是 |
| 手机号 | 是 | 手机号码 | 11位数字，唯一 | 是 | 是 |
| 邮箱 | 否 | 邮箱地址 | 邮箱格式 | 是 | 是 |
| 账号状态 | 是 | 账号启用状态 | 开关 | 是 | 是 |
| 归属机构与角色 | 是 | 用户归属机构及角色 | 至少一个 | 是 | 是 |

### 7.5 交互规则

| 操作 | 行为 |
|------|------|
| 点击保存 | 校验通过后保存，关闭弹窗，刷新列表 |
| 点击取消 | 关闭弹窗，不保存 |
| 点击添加归属机构 | 弹出机构选择弹窗 |
| 点击编辑 | 编辑当前归属机构配置 |
| 点击删除 | 移除当前归属机构 |
| 校验失败 | 显示错误提示，焦点定位到错误字段 |
