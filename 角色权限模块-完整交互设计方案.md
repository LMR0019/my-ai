## 1. 功能概述

角色权限模块用于管理系统角色和权限配置。模块包含：

**列表页** — 角色管理入口：
- 查看所有角色列表
- 统计各端口角色数量
- 搜索筛选角色
- 启用/禁用角色
- 快速操作入口

**新建/编辑角色弹窗** — 角色信息管理：
- 角色基本信息
- 权限配置（权限树选择）
- 数据权限范围配置

**分配用户弹窗** — 用户角色分配：
- 选择目标机构
- 分配/移除角色用户

**查看用户弹窗** — 查看角色用户：
- 查看拥有此角色的用户列表

**复制角色弹窗** — 快速创建：
- 基于现有角色创建副本

| 字段 | 必填 | 说明 | 校验规则 | 新建 | 编辑 |

| 角色名称 | 是 | 角色显示名称 | 最大50字符 | 是 | 是 |

| 备注/描述 | 否 | 角色说明 | 最大200字符 | 是 | 是 |
| 权限配置 | 是 | 角色功能权限 | 权限树选择 | 是 | 是 |
| 数据权限范围 | 是 | 数据访问范围 | 单选 | 是 | 是 |

### 7.4 权限配置区

| 操作 | 行为 |
|------|------|
| 选择所属端口 | 权限树自动聚焦该端口权限 |
| 点击端口Tab | 筛选显示该端口权限 |
| 点击全选 | 选中当前Tab下所有权限 |
| 点击清空 | 取消选中当前Tab下所有权限 |
| 点击父节点 | 全选/取消子节点 |

### 7.5 交互规则

| 操作 | 行为 |
|------|------|
| 点击保存 | 校验通过后保存，关闭弹窗，刷新列表 |
| 点击取消 | 关闭弹窗，不保存 |
| 校验失败 | 显示错误提示，焦点定位到错误字段 |
