## 1. 功能概述

菜单管理模块用于管理系统导航菜单。模块包含：

**列表页** — 菜单管理入口：
- 端口Tab筛选不同端菜单
- 搜索筛选菜单
- 统计各类型菜单数量
- 树形结构展示菜单层级
- 拖拽调整菜单顺序

**新建/编辑弹窗** — 菜单配置：
- 菜单基本信息
- 上级菜单选择
- 菜单类型和路由配置

**同步权限树** — 权限同步：
- 同步菜单与权限关联

## 6. 树形表格

### 6.1 表格列定义

| 列号 | 列名 | 宽度 | 内容 | 对齐 |
|------|------|------|------|------|
| 1 | ▶/▼ | 40px | 展开/折叠 | 居中 |
| 2 | 菜单名称 | 自适应 | 菜单图标+名称 | 左对齐 |
| 3 | 菜单编码 | 180px | 编码 | 左对齐 |
| 4 | 菜单类型 | 80px | 类型标签 | 居中 |
| 5 | 状态 | 80px | 状态标签 | 居中 |
| 6 | 操作 | 200px | 编辑/新增子菜单/删除/排序 | 居中 |

### 6.2 各列详细设计

#### 列1：展开/折叠

| 状态 | 图标 | 说明 |
|------|------|------|
| 展开 | ▼ | 有子菜单，已展开 |
| 折叠 | ▶ | 有子菜单，已折叠 |
| 无子菜单 | — | 灰色占位 |

#### 列2：菜单名称

```
📁 系统设置
```

| 字段 | 说明 |
|------|------|
| 图标 | 根据菜单类型显示不同图标 |
| 名称 | 菜单显示名称 |

#### 列4：菜单类型

| 类型 | 标签样式 |
|------|---------|
| 目录 | 灰色标签 |
| 菜单 | 蓝色标签 |
| 按钮 | 绿色标签 |

#### 列5：状态

| 状态 | 标签颜色 | 交互 |
|------|---------|------|
| 启用 | 绿色 | 点击可切换为禁用 |
| 禁用 | 灰色 | 点击可切换为启用 |

#### 列6：操作

| 按钮 | 显示条件 | 交互 |
|------|---------|------|
| 编辑 | 所有菜单 | 打开编辑弹窗 |
| 新增子菜单 | 菜单层级<3 | 打开新建弹窗（带父菜单） |
| 删除 | 无子菜单的菜单 | 弹出删除确认弹窗 |
| 排序 | 有子菜单的目录 | 拖拽手柄 |

### 6.3 树形交互

| 操作 | 行为 |
|------|------|
| 点击展开箭头 | 展开/折叠子菜单 |
| 拖拽排序 | 调整同级菜单顺序 |
| 行悬浮 | 高亮显示 |

```
└─────────────────────────────────────────────────────────────┘
```

### 8.3 字段说明

| 字段 | 必填 | 说明 | 校验规则 | 新建 | 编辑 |
|------|------|------|---------|------|------|
| 菜单名称 | 是 | 菜单显示名称 | 最大50字符 | 是 | 是 |
| 上级菜单 | 是 | 父级菜单 | 下拉选择 | 是 | 否 |
| 菜单类型 | 是 | 目录/菜单/按钮 | 下拉选择 | 是 | 否 |
| 菜单编码 | 是 | 系统唯一标识 | 英文数字，唯一 | 是 | 否 |
| 图标 | 否 | 菜单图标 | 图标选择器 | 是 | 是 |
| 路由地址 | type=MENU时必填 | 页面路由路径 | 以/开头 | 是 | 是 |
| 状态 | 是 | 菜单启用状态 | 开关 | 是 | 是 |

### 8.4 菜单类型联动

| 选择类型 | 路由地址 |
|----------|---------|
| 目录 | 隐藏，无需填写 |
| 菜单 | 显示，必填 |
| 按钮 | 隐藏，无需填写 |

### 8.5 交互规则

| 操作 | 行为 |
|------|------|
| 点击保存 | 校验通过后保存，关闭弹窗，刷新列表 |
| 点击取消 | 关闭弹窗，不保存 |
| 选择菜单类型 | 联动显示/隐藏路由地址字段 |
| 校验失败 | 显示错误提示，焦点定位到错误字段 |
